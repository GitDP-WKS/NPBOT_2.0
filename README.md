# РЭС AI 2.0

Чистая версия системы определения филиала и РЭС по адресам Республики Татарстан.

## Основные принципы

- Старые таблицы РЭС AI 1 не изменяются.
- Новая версия использует таблицы с префиксом `res_ai_v2_`.
- Непротиворечивая запись источника не считается подтвержденной человеком.
- Уникальное название в одном адресном контексте получает расчетное доверие до 99,9%.
- Если одинаковое название встречается в двух разных контекстах, доверие без района составляет около 50%; в трех — около 33,3%.
- Район, СНТ и улица уточняют контекст и позволяют принять окончательное решение.
- Филиал всегда определяется по официальному РЭС.
- Администратор применяет решение сразу. Обычное решение применяется после трех одинаковых голосов.
- Все действия записываются в журнал. Администратор может отменить принятое решение.
- Модель обучается на однозначных адресах и размеченных реальных текстах, но конфликты и многозначные адреса исключаются.
- Новая модель не публикуется, если ухудшает общую точность, Macro-F1 или качество отдельного РЭС.

## Запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Secrets

```toml
ADMIN_PASSWORD = "ваш-пароль"
DATABASE_URL = "postgresql://USER:PASSWORD@HOST/DATABASE?sslmode=require"
```

Используется тот же проект Neon. Приложение само создает таблицы `res_ai_v2_*`. Старые таблицы остаются без изменений.

## Первый запуск

1. Развернуть новый репозиторий в Streamlit Community Cloud.
2. Скопировать прежние `ADMIN_PASSWORD` и `DATABASE_URL` в Secrets.
3. Войти как администратор.
4. Открыть `Настройки`.
5. Нажать `Импортировать данные старой версии`.
6. Открыть `Анализ и обучение` и запустить `Полный цикл`.
7. Проверить показатели кандидата и опубликовать его, если порог качества пройден.

## Разделы

Обычному проверяющему доступны:

- `Проверка`;
- `Определение`.

Администратору доступны:

- `Главная`;
- `Проверка`;
- `Определение`;
- `Загрузка`;
- `База знаний`;
- `Анализ и обучение`;
- `Качество`;
- `Журнал`;
- `Настройки`.

## Программный интерфейс

```bash
uvicorn res_ai_v2.api:app --host 0.0.0.0 --port 8000
```

- `POST /predict`

```json
{

}
```
